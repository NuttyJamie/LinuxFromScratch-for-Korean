6.42. XML::Parser-2.46

XML::Parser 모듈은 James Clark의 XML 파서인 Expat에 대한 Perl 인터페이스이다.

예상 빌드 시간: 0.1 SBU 이하
필요 디스크 공간: 2.4 MB

6.42.1. XML::Parser 설치

XML::Parser 컴파일을 준비한다: 

perl Makefile.PL

패키지를 컴파일한다: 

make

결과를 테스트하려면 실행하라: 

make test

패키지를 설치한다: 

make install
