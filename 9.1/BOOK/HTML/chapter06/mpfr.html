6.20. MPFR-4.0.2

MPFR 패키지는 다중 정밀 산술 기능을 포함하고 있다.

예상 빌드 시간: 0.8 SBU
필요 디스크 공간: 37 MB

6.20.1. MPFR 설치

MPFR 컴파일을 준비한다: 

./configure --prefix=/usr        \
            --disable-static     \
            --enable-thread-safe \
            --docdir=/usr/share/doc/mpfr-4.0.2

패키지를 컴파일하고 HTML 문서를 생성한다: 

make
make html

중요

이 절의 MPFR 테스트 스위트는 매우 중요하다. 어떤 상황에서도 건너뛰지 않도록 하라.

결과를 테스트하고 모든 테스트가 통과했는지 확인하라: 

make check

패키지와 문서를 설치하라: 

make install
make install-html
