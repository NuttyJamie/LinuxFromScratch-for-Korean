6.17. Bc-2.5.3

Bc 패키지는 임의의 정밀 숫자 처리 언어를 포함하고 있다.

예상 빌드 시간: 0.1 SBU
필요 디스크 공간: 2.9 MB

6.17.1. Bc 설치

Bc 컴파일을 준비한다: 

PREFIX=/usr CC=gcc CFLAGS="-std=c99" ./configure.sh -G -O3

Configure 옵션들의 의미:

CC=gcc CFLAGS="-std=c99"

사용할 컴파일러 및 C 표준을 지정한다.

-O3

사용할 최적화 단계를 지정한다.

-G

GNU bc가 없으면 작동하지 않는 테스트 스위트의 일부를 생략한다.

패키지를 컴파일한다: 

make

bc를 테스트하려면 실행하라: 

make test

패키지를 설치한다: 

make install
