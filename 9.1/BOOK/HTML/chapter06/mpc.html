6.21. MPC-1.1.0

MPC 패키지에는 임의의 높은 정밀도를 갖는 복잡한 숫자의 계산과 정확한 반올림을 위한 라이브러리를 포함하고 있다.

예상 빌드 시간: 0.3 SBU
필요 디스크 공간: 22 MB

6.21.1. MPC 설치

MPC 컴파일을 준비한다: 

./configure --prefix=/usr    \
            --disable-static \
            --docdir=/usr/share/doc/mpc-1.1.0

패키지를 컴파일하고 HTML 문서를 생성하라: 

make
make html

결과를 테스트하려면, 다음을 실행하라: 

make check

패키지와 문서를 설치하라: 

make install
make install-html

설치된 라이브러리들: libmpc.so
