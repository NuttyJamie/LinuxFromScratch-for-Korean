6.34. Grep-3.4

Grep 패키지는 파일 내용을 검색하는 프로그램을 포함하고 있다.

예상 빌드 시간: 0.7 SBU
필요 디스크 공간: 39 MB

6.34.1. Grep 설치

Grep 컴파일을 준비한다: 

./configure --prefix=/usr --bindir=/bin

패키지를 컴파일한다: 

make

결과를 테스트하려면 실행하라: 

make check

패키지를 설치한다: 

make install

6.34.2. Grep의 내용

설치된 프로그램들: egrep, fgrep, grep

요약

egrep

확장 정규 표현식으로 검색된 라인 출력

fgrep

고정 문자열 목록으로 검색된 라인 출력

grep

일반 정규 표현식으로 검색된 라인 출력
