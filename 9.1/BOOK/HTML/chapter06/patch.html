6.68. Patch-2.7.6

Patch 패키지에는 일반적으로 diff 프로그램에서 만든 “patch” 파일을 적용하여 파일을 수정하거나 만드는 프로그램이 포함되어 있다.

예상 빌드 시간: 0.2 SBU
필요 디스크 공간: 13 MB

6.68.1. Patch 설치

Patch 컴파일을 준비한다: 

./configure --prefix=/usr

패키지를 컴파일한다: 

make

결과를 테스트하려면 실행하라: 

make check

패키지를 설치한다: 

make install

6.68.2. Patch의 내용

설치된 프로그램: patch

요약

patch

패치 파일에 따라 파일 수정 [패치 파일은 보통 diff 프로그램으로 작성된 차이점 목록이다. 이러한 차이점을 원본 파일에 적용하면 patch가 패치된 버전을 생성한다.]
